# Street-Vendor-Digitalization
Built on IBM watsonx Orchestrate and Granite models, this multi-agent AI empowers street vendors by simplifying PM SVANidhi micro-loans via a RAG pipeline. It offers step-by-step, jargon-free guidance for merchant UPI setup, e-KYC, and map pinning via a zero-friction mobile web chat, actively driving financial inclusion for micro-entrepreneurs.
